# Systematic Review Checklist (PRISMA 2020)

## Phase 1: Planning
- [ ] Define research question (PICO)
- [ ] Determine eligibility criteria (inclusion/exclusion)
- [ ] Register protocol (PROSPERO, OSF, or other)
- [ ] Assemble review team (minimum 2 independent screeners)
- [ ] Define outcomes and prioritization
- [ ] Plan data extraction forms
- [ ] Plan risk of bias assessment tool (RoB 2, ROBINS-I, Newcastle-Ottawa)
- [ ] Plan synthesis method (narrative, meta-analysis, network meta-analysis)
- [ ] Plan certainty of evidence assessment (GRADE)

## Phase 2: Search
- [ ] Design search strategy for each database (PubMed/MEDLINE, Embase, Cochrane CENTRAL, Web of Science, Scopus)
- [ ] Include both controlled vocabulary (MeSH, Emtree) and free-text terms
- [ ] Apply date and language restrictions if justified
- [ ] Search clinical trial registries (ClinicalTrials.gov, WHO ICTRP)
- [ ] Search grey literature (conference abstracts, dissertations, regulatory documents)
- [ ] Contact experts and study authors for unpublished data
- [ ] Document exact search strings and date of search

## Phase 3: Screening
- [ ] Import citations into reference manager (EndNote, Zotero, Covidence, Rayyan)
- [ ] Remove duplicates
- [ ] Title/abstract screening by 2 independent reviewers
- [ ] Full-text screening by 2 independent reviewers
- [ ] Document reasons for exclusion at full-text stage
- [ ] Calculate inter-rater agreement (Cohen kappa)
- [ ] Resolve disagreements by consensus or third reviewer

## Phase 4: Data Extraction
- [ ] Pilot test extraction form on 2-3 studies
- [ ] Extract by 2 independent reviewers
- [ ] Extract study characteristics, participant characteristics, interventions, outcomes
- [ ] Extract effect estimates with confidence intervals
- [ ] Contact authors for missing data
- [ ] Assess risk of bias for each included study

## Phase 5: Synthesis
- [ ] Assess clinical and methodological heterogeneity
- [ ] Choose effect measure (RR, OR, HR, MD, SMD)
- [ ] Choose meta-analysis model (fixed-effect vs random-effects, typically DerSimonian-Laird)
- [ ] Assess statistical heterogeneity (I², tau², chi² test)
- [ ] Explore sources of heterogeneity (subgroup analyses, meta-regression)
- [ ] Assess small-study effects (funnel plot, Egger test)
- [ ] Perform sensitivity analyses
- [ ] Assess certainty of evidence using GRADE

## Phase 6: Reporting
- [ ] Follow PRISMA 2020 checklist (27 items + 4 for abstracts)
- [ ] Create PRISMA flow diagram
- [ ] Report funding sources and conflicts of interest
- [ ] Make data available (supplementary materials, data repository)
- [ ] Register updates if protocol changes
