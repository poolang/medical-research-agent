# Medical Research Agent Framework

An open-source skill system for healthcare professionals to conduct evidence-based medical research.

## Core Modules
1. PICO Builder
- Literature Search
- Evidence Grading
- Systematic Review Assistant
- Trial Matcher
- Guideline Tracker
- Drug Interaction Researcher
- Case Report Builder

## Verification Layer
- Every claim requires PMID citation
- Auto-flag industry-funded trials
- Retraction watch via PubMed
- Human attending sign-off for "vetted" status

## Usage
Install via: `bankr install_skill("https://github.com/poolang/medical-research-agent")`
